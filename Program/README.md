# GPU Market Analytics

Цей проєкт — веб-застосунок на Python для аналізу ринку відеокарт. Він допомагає переглядати історичні ціни, порівнювати характеристики сучасних GPU, будувати прості прогнози цін та вивчати частки ринку виробників.

## Що вміє застосунок
- аналіз історичних цін GPU;
- перегляд сучасних характеристик і цін моделей;
- прогнозування тенденцій на основі історичних даних;
- візуалізація поставок і ринкових часток;
- перегляд сирих таблиць з даними.

## Структура проєкту
- `app.py` — головний файл застосунку
- `requirements.txt` — залежності проєкту
- `data/` — CSV-файли з даними

## Установка
1. Переконайтесь, що встановлено Python 3.9–3.13.
2. Встановіть залежності:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустіть застосунок:
   ```bash
   streamlit run app.py
   ```
   Якщо з’являється запит Email — просто натисніть Enter.
4. Відкрийте сторінку:
   ```text
   http://localhost:8501
   ```

## Використання
У застосунку доступні вкладки:
- Market overview — сучасні GPU та ціни
- Price dynamics & forecast — історія та прогнозування
- Shipments & market share — частки ринку
- US price index — індекс цін
- Data — перегляд вихідних таблиць
